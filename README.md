Aplicativo Android que exibe sinais vitais simulados e registro de sintomas. 

Exibe frequência cardíaca e passos com dados mockados.
Atualização contínua usando Coroutines e Flow.


Sintomas:
Permite registrar sintomas e visualizar histórico.
Dados persistidos com Room, mantém histórico após fechar e abrir o aplicativo novamente.


Tecnologias:
Kotlin
Fragments
XML (findViewById)
ViewModel
LiveData
Coroutines
Flow
Room (Entity/DAO/DB)
MVVM
Repository

Como rodar:
Abrir no Android Studio
Gradle Sync
Run em emulador ou dispositivo
